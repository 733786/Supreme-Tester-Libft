## Installation

The folder name must necessarily be called "libft"
```sh
sh tester.sh
vim libft-war-machine/my_config.sh
```
then change the name of the 8ยบ row from

```sh
PATH_LIBFT="../"
```
to
```sh
PATH_LIBFT="Users/{user}/{path}/libft"
```
you can find the right path with
```sh
pwd
```
